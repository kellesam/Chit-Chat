What is Chit Chat?

Chit Chat is a simple, lightweight chat client so you can worry less about users and logins and focus on what matters: chatting!


How do I start chatting?

Create a new room above and choose a name. (Private rooms coming soon!)


How do I invite others to a chat?

Users can join a chat room by going directly to an open room's url, or by using the join form above with a valid room id.


What about all of my other questions?

This is an ongoing open-source project from students at Oregon State University, and will be updated periodically. Check the github repository for more info!
